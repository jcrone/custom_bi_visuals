Visual 1 — Single KPI Card

Displays a single KPI metric with a large value, percentage variance against a target, and an optional sparkline trend line. Ideal for highlighting one key metric per card — drop multiple cards side-by-side to build a KPI row.

TargetMeasureComparison value — variance is calculated as (Value - Target) / Target
TrendMeasureValues plotted on the sparkline (e.g. Revenue by month)

Visual 3 — Date Slicer / Date Picker

A full-featured date slicer with a calendar picker, preset quick-filters (Yesterday, This Week, Last Month, This Year, etc.), custom "N days" inputs, and single-date or date-range modes. Supports two display modes: Expanded (inline calendar) and Compact (pill that opens a modal dialog). Filters all other visuals on the report page.

Data Fields
FieldTypeDescription

Visual 4 — Radial Gauge

A modern radial/arc gauge showing progress toward a goal with animated fill, configurable threshold bands (danger/warning/good), a target needle marker, and variance display. Ideal for quota attainment, budget utilization, SLA compliance, or any metric with a defined min/max range.

Data Fields
FieldTypeDescription
ValueMeasureThe current value displayed on the gauge
TargetMeasureTarget/goal — shows a needle marker and variance
Max ValueMeasureMaximum scale value (defaults to 100)
Min ValueMeasureMinimum scale value (defaults to 0)
Formatting Options
SettingOptionsDescription
Arc ColorColor pickerFill color of the progress arc
Arc Thickness8 – 60Width of the arc stroke
Start / End Angle0° – 720°Geometry of the arc sweep (default 135°–405° for ¾ gauge)
ThresholdsOn / OffShow danger/warning/good color bands behind the arc
Display UnitsNone / K / M / AutoAbbreviate large numbers

Visual 5 — Bump Chart

Shows how rankings change over time — each entity is a line, and the Y-axis shows rank position (#1, #2, #3...) rather than raw value. Ideal for competitive comparisons like sales rep rankings, product performance shifts, or regional standings across time periods.

Data Fields
FieldTypeDescription
AxisGroupingTime periods for the x-axis (e.g. Month, Quarter)
LegendGroupingEntities to rank (e.g. Product, Region, Sales Rep)
ValuesMeasureMetric used to compute rankings each period
Formatting Options
SettingOptionsDescription
Line Width1 – 8Thickness of the rank lines
Smooth CurvesOn / OffS-shaped bump curves vs straight lines
Dim Opacity5% – 100%How faded non-hovered lines become
LabelsStart / End / BothShow entity names at line endpoints
Data PointsOn / Off, 2 – 12Show dots at rank positions

Visual 6 — Pulse Map

An EKG/heartbeat-style chart where each entity gets its own horizontal trace with sharp pulse spikes proportional to value. Instantly reveals who's active, who's flat, and the rhythm of activity across time.

Data Fields
FieldTypeDescription
EntityGroupingEach entity gets its own heartbeat row (e.g. Sales Rep, Server)
AxisGroupingTime periods along the trace (e.g. Week, Month)
ValuesMeasurePulse spike height — larger values produce taller spikes
Visual 7 — Hive Plot

A radial connection web with two axes — source entities on the left, destinations on the right — connected by curved arcs whose thickness represents weight. Reveals relationship patterns that tables completely miss.

Data Fields
FieldTypeDescription
SourceGroupingSource entities on the left axis (e.g. Teams, Departments)
DestinationGroupingDestination entities on the right axis (e.g. Projects, Products)
WeightMeasureConnection strength — thicker arcs for larger values
Visual 8 — Ridge Chart

Stacked overlapping area ridges — each row is a series rendered as a smooth terrain shape, layered with vertical offset like mountain ranges. Inspired by Joy Division's Unknown Pleasures album art. Includes a dark mode toggle for the classic look. Great for comparing distribution shapes across categories.

Data Fields
FieldTypeDescription
SeriesGroupingEach series becomes a ridge row (e.g. Year, Product)
AxisGroupingX-axis within each ridge (e.g. Month, Day)
ValuesMeasureRidge height — plotted as the area shape
Visual 9 — Drilldown Donut

A ZoomCharts-style drilldown donut chart that shows one hierarchy level at a time. Click a segment to smoothly morph arcs and reveal its children. Click the center to navigate back up. Supports up to three hierarchy levels with animated transitions and breadcrumb navigation.

Data Fields
FieldTypeDescription
Level 1GroupingTop-level categories shown on first view
Level 2GroupingChildren revealed when drilling into a Level 1 segment
Level 3GroupingOptional third level for deeper drill-down
ValuesMeasureArc sizing — determines segment proportions
