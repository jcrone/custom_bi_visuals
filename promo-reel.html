Visual 01

Pulse Map

EKG-style heartbeat traces where each entity pulses with activity. Big spikes mean big values. Flatlines mean silence. Instantly spot who's performing and the rhythm of their output.

  • Entity Each row is an entity trace
  • Axis Time periods along the pulse
  • Values Spike height = value magnitude
Visual 02

Hive Plot

A radial connection web with two axes — sources on the left, destinations on the right. Curved arcs reveal weighted relationships that tables and bar charts completely miss.

  • Source Origin entities on left axis
  • Destination Target entities on right axis
  • Weight Arc thickness = connection strength
Visual 03

Ridge Chart

Overlapping terrain ridges inspired by Joy Division's iconic album art. Each series becomes a smooth mountain shape layered front-to-back. Includes dark mode for the classic look.

  • Series Each ridge is a category row
  • Axis X-axis points within each ridge
  • Values Ridge height = area shape
